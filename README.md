# ✨ What is InventoryScope?

InventoryScope is a simple desktop application that helps Roblox creators browse and organize **their own assets** in one place.

No browser tabs.

No clutter.

Just a clean desktop app.

✅ Browse your assets

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/inventoryscope.git
```

Go to the project

```bash
cd inventoryscope
```

Install packages

```bash
npm install
```

Run

```bash
cargo tauri dev
```
